# gearai

### About: 
  GearAI's Python Library for solving various AI needs.

### Installation: 
To install, Open the Command Prompt (or) Terminal and type 
```shell
pip install gearai
``` 
If pip doesn't work, try using 
```shell
pip3 install gearai
``` 

### Usage: 
To import gearai library, add the line 
```python
import gearai
```
